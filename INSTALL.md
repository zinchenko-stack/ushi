# Установка ushi

## Первый запуск

1. Скачай **ushi.dmg** (или **ushi.zip**) по ссылке, которую тебе прислали.
2. Открой DMG → перетащи **ushi** в папку **Программы** (Applications).
3. Открой Launchpad или Finder → Программы → найди **ushi**.

### macOS скажет «приложение от неустановленного разработчика»

Это нормально — приложение распространяется без сертификата Apple ($99/год — пока не оплачено). Чтобы открыть:

**macOS 15 Sequoia и новее:**
1. Зайди в **Системные настройки → Конфиденциальность и безопасность**.
2. Прокрути вниз — увидишь сообщение про ushi.
3. Нажми **«Открыть всё равно»** → подтверди паролем.

**Альтернатива (macOS любой версии):**
1. В Finder найди **ushi** в Программах.
2. **Правый клик на иконке → «Открыть»** (именно через правый клик, не двойным).
3. В появившемся диалоге нажми **«Открыть»**.
4. Дальше можно запускать обычным двойным кликом.

Это надо сделать **только один раз** после установки.

## Разрешения

При первом запуске система попросит два разрешения:

- **Запись экрана** — для записи звука из созвонов (приложение слышит то же, что и динамики). Без него запись не будет работать. После выдачи приложение перезапустится — это нормально, нужно один раз.
- **Микрофон** — для записи твоего голоса. Если выключишь микрофон в настройках приложения, разрешение не нужно.

Эти разрешения можно изменить в **Системные настройки → Конфиденциальность и безопасность**.

## Обновления

Приложение само проверяет наличие новой версии при запуске. Если есть обновление — внизу появится плашка «Доступна версия X · Скачать».

Чтобы обновиться:

1. Нажми **Скачать** в плашке (или выбери в меню **ushi → Проверить обновления…**).
2. Откроется ссылка на новый DMG — скачай его.
3. Закрой текущую версию приложения.
4. Открой DMG → перетащи новую версию в Программы (с заменой).
5. Запусти — снова **правый клик → Открыть** при первом старте новой версии (потому что обновление = «новый» бинарник для macOS).

## Если что-то сломалось

- **Приложение не запускается:** проверь, что в Системных настройках → Конфиденциальность → Запись экрана для ushi включена галка. Иногда после обновления её нужно выдать заново.
- **Не пишется звук собеседника:** значит, нет доступа к записи экрана. Включи в Системных настройках и перезапусти приложение.
- **Не слышно твой голос в записи:** в самом приложении внизу есть переключатель «Микрофон выключен / включён» — проверь, что включён.

## Поддерживаемые версии macOS

- **macOS 14.6 Sonoma** и новее.
- Захват микрофона работает с **macOS 15 Sequoia** и новее. На macOS 14 запишется только системный звук (голос собеседника), твой голос — нет.
