# Установка ushi

## Первый запуск

1. Скачай **ushi.dmg** по ссылке, которую тебе прислали.
2. Открой DMG → перетащи **Ushi** в папку **Programs** (Applications).
3. Открой Ushi из Applications (через Launchpad или Finder → Программы).

### macOS заблокирует первый запуск

Это нормально — приложение распространяется без сертификата Apple. Окно скажет «не удалось проверить разработчика».

1. Нажми **«Готово»** в окне блокировки.
2. **Системные настройки → Конфиденциальность и безопасность**.
3. Прокрути вниз — увидишь сообщение про Ushi.
4. Нажми **«Открыть всё равно»** → введи пароль → подтверди **«Открыть»**.

Это нужно сделать **только один раз** после установки.

## Подготовка приложения (один раз)

При первом запуске Ushi скачает модель распознавания речи (~1.5 ГБ). Откроется окно **«Подготовка Ushi»** с прогресс-баром.

Можно нажать **«Свернуть и пользоваться сейчас»** — записи будут работать сразу, а транскрипция начнётся автоматически как только модель догрузится. В основном окне сверху будет узкая плашка с прогрессом.

Скачивание возобновляется при обрыве сети. Если закроешь приложение посередине — продолжит с того же места при следующем запуске.

## Разрешения

При первой записи система попросит два разрешения:

- **Запись экрана** — для захвата системного звука (голос собеседника в созвоне). Без этого приложение не запишет то, что слышишь в наушниках.
- **Микрофон** — для записи твоего голоса. Если выключишь микрофон в Ushi (тоггл в окне записи), это разрешение не нужно.

После выдачи прав приложение автоматически перезапустится.

## Обновления

Ushi сама проверяет наличие новой версии при запуске. Внизу окна появится плашка «Доступна версия X · Скачать».

Чтобы обновиться:

1. Нажми **«Скачать»** в плашке (или в меню **Ushi → Проверить обновления…**).
2. Скачается новый DMG.
3. Закрой текущую версию приложения.
4. Открой DMG → перетащи новую версию в Applications с заменой.
5. Запусти → снова **«Открыть всё равно»** через настройки при первом старте (для macOS новый бинарь = «новое» приложение).

Скачанная модель распознавания **переустанавливать не надо** — она хранится в `~/.ushi/models/` и продолжит работать с новой версией.

## Если что-то сломалось

- **Приложение не открывается:** проверь, что в **Конфиденциальность → Запись экрана** для Ushi включена галка. После обновлений macOS её иногда нужно выдать заново.
- **Не пишется звук собеседника:** доступ к Записи экрана отключён или требует повторной выдачи.
- **Не слышно твой голос в записи:** проверь тоггл «Микрофон» в Ushi.
- **Модель не качается:** проверь интернет, нажми «Повторить» на онбординге или клик «Подробнее» на плашке → «Повторить».
- **Транскрипция падает с ошибкой:** возможно, модель скачалась битая. Удали файл `~/.ushi/models/ggml-large-v3-turbo.bin` (через Finder: Cmd+Shift+G → `~/.ushi/models/`) и перезапусти Ushi — скачивание начнётся заново.

## Поддерживаемые версии macOS

- **macOS 14.6 Sonoma** и новее.
- Захват микрофона работает с **macOS 15 Sequoia** и новее. На macOS 14 запишется только системный звук (голос собеседника), твой голос — нет.
